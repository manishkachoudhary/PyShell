PyShell is a custom command-line shell developed in Python that integrates
Operating System commands, File Handling, and Data Structure operations
into a single interactive environment.

## Features
- Custom shell interface (PyShell>)
- Stack, Queue, Linked List, Circular List, Doubly Linked List operations
- File handling (read, write, append, delete)
- PDF file read and write support
- DOCX file read and write support
- Directory navigation (cd, pwd)
- Executes system commands
- Color-coded output for better readability

## Data Structures Implemented
- Stack
- Queue
- Singly Linked List
- Circular Linked List
- Doubly Linked List

## Technologies Used
- Python
- Data Structures
- File Handling
- OS & Subprocess modules

## External Libraries
- colorama
- fpdf
- PyMuPDF (pymupdf)
- python-docx
